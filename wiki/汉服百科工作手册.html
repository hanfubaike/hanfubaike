# 汉服百科工作手册

# 导言

为了使汉服百科早日成长起来，成为能够帮助每一位朋友解答疑惑的平台，汉服百科需要你的帮助！


首先，你需要加入小组。


现阶段，您可以为我们做如下工作：

  • 添加新的词条
  • 为现有词条增加内容
  • 提出意见和建议

您也可以参考我们的“词条样本”进行进一步的编写。


# 添加新的词条

您可以打开汉服百科知识库，点击右上角的新建按钮，选择从模板新建，然后选择词条模板，开始撰写新的词条内容。稍后热心的百科编辑会对它进行进一步的编辑。可能是添加链接、丰富内容，也可能是和现有的内容进行对接整合。


词条应当尽量采用客观理性的学术语言，避免第一人称和个人情绪化的语言。您的名字将永远作为汉服百科编纂者，留在百科系统中。


# 为现有词条增加内容

当您感到词条不完整、不准确时，可以点击“编辑”，修改现有的词条。


# 如何判断词条是否完整？

一个完整的词条应具备如下内容：文字部分、参考文献。


# 如何编写文字部分

服饰形制词条的文字部分，应至少包括定义、主要种类、结构特征、文物依据四部分。对于文物依据尚不充分、或形状较为多样的形制，还应当补充文字形态的文献资料。


汉服术语词条的文字部分，应力求将术语的定义说清楚，使零基础的读者也一看就能明白。


词条是现代语词，还是传统语词，应当尽量予以说明；当各家对于同一个词持不同理解时，应在确保客观公正的基础上，将各家说法都予以陈述。


与词条相关的常见误区、争议也应当在词条中如实记述。


插入恰如其分的图片，可以帮助访问者理解词条的内容，因此我们鼓励为现有的词条增加图片。


对词条的修改，请务必在修改处注明依据与参考文献，方便今后的进一步核查和编辑。


# 注明出处

汉服百科项目采用知识共享协议——署名 - 相同方式共享（BY-SA），使用者可以对本项目的内容进行转载、节选、混编、二次创作，可以将其运用于商业用途，唯须署名作者和出处，并且采用本创作的内容必须同样采用本协议进行授权。具体见：此处为语雀文档，点击链接查看：https://www.yuque.com/hanfubaike/wiki/uyfgbv。


同样的，如果你在词条中引用了其他人的创作（图片、文字、视频等），请务必注明出处，如有必要，请联系原作者取得授权。
