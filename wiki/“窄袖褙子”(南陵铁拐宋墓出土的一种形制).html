# “窄袖褙子”(南陵铁拐宋墓出土的一种形制)

# 基本信息

  • 名称：“窄袖褙子”(南陵铁拐宋墓出土的一种形制)
  • 别名：窄袖褙子（考古简报中使用的名称）、“飞机袖”（俗称，因为在时装中有飞机袖这种袖型）“蝙蝠袖”（俗称，因为在时装中有蝙蝠袖这种袖型）、“抱抱袖”（部分汉服爱好者起的爱称）（以上三种称呼衫/袄经常被省略）、宋制直领对襟窄袖短衫/袄（被包含）
  • 分类：应是上衣，在宋制着装系统的层次位置暂不明确，暂定为服-体衣-外衣（？）-襦（衣）
  • 特征：短衣身，对襟，长袖，袖根宽，袖口窄，腋下开胯，轮廓平直
  • 流行年代：北宋徽宗政和年间
  • 文物依据：安徽南陵铁拐北宋墓出土（孤证）

# 定义

是以安徽南陵铁拐宋墓出土的直领对襟窄袖短上衣为原型的一种形制，考古报告中称为“窄袖褙子”。


褙子是一个宋代真实存在，但在汉服界定义有争议的形制概念，受此影响，一些人倾向于避免使用“窄袖褙子”这一称呼。但目前关于此形制在学术界暂未见到详细讨论，为避免无意义的争论、误导和混乱，本wiki仍然采用考古报告中的名称，使用“窄袖褙子”（南陵铁拐宋墓出土的一种形制）作为本形制的暂定名。


由于袖根宽阔，袖口窄小，形如现代时装中的“飞机袖”或“蝙蝠袖”，因而又俗称“飞机袖”衫/袄或“蝙蝠袖”衫/袄，因为这种称呼过于现代，故而有人坚决反对。


此形制适合的穿着场合（大礼服、常服或便服）与层次位置尚不明确。汉服商家一般作为夏季外衣推出。


# 结构特征

直领对襟短上衣，具有独特的窄袖，袖根、袖口数据相差颇大。衣身轮廓平直，两边开衩。


这种形制的文物依据，目前掌握的仅有孤证。仅凭考古简报照片很难看出衣料层次。但在一则新闻中出现了形制相似的文物照片，据新闻，是双层的[1] ，装在随葬包袱内，不知是否同一件。目前商家通常制作成单层。为了适应不同的季节和天气，同一种形制的汉服往往存在单、夹衣两种厚度，所以，虽然缺少旁证，但按照常理推断，该形制的单层（衫）或双层（袄）应该都是成立的。


# 艺术表现

白沙宋墓墓室壁画中表现的褙子往往衣袖宽松，“袖腋部分肥阔，愈近腕部愈收缩，至腕部以一道紧裹手腕的袖圈结束”。孟晖将之形容为类似汉代的“大掖衣”或时装中的“蝙蝠袖”[2] ，疑似本词条所述袖型的艺术表现，而衣长又有显著不同。


大掖衣在服饰史术语中是指一种先秦既有形制，“大掖”的所指在学界有争论，故而本wiki不采用“大掖某”称呼此中形制。详见“大掖衣”词条。

白沙宋墓壁画“开芳宴”中窄袖褙子的表现



# 文物依据

此形制目前掌握的公开资料中仅有孤证，为安徽南陵铁拐宋墓出土。


直领，对襟，腋下开胯，短衣身，长袖，袖口窄收。


衣长64.5cm，肩宽39cm，袖长44.5cm，袖宽12.2～37cm。


墓主人：安康郡太君（一名朝廷命妇）


此墓为北宋徽宗政和年间墓（北宋晚期）。[3]


在人民网安徽频道对该墓的报道中，还出现了一张形制相似的文物照片，据报道推测，出土于该墓随葬包袱中，双层、丝质，薄如蝉翼，十分精美。由于缺少详细资料，暂不清楚与《安徽南陵铁拐宋墓发掘简报》图版中的“窄袖褙子”是否同一件文物。


文物肩宽与衣长数据与图片直观反映的衣身形状比例不一致，暂不明确其中原因。商家据此文物仿制时，通常会将39cm视为半肩宽，再在此基础上进行调整。


# 争议


# “窄袖褙子”的袖称呼为“飞机袖”或“蝙蝠袖”是否合适

一些人认为此形制的俗名“飞机袖”或“蝙蝠袖”属于戏称，气息过于现代，不适合用作传统汉服的称呼。[4][5]


微博用户@浅夏草染 和@狐鸦youko 认为可以用“箭袖”称呼这种袖型。@浅夏草染 说以前就用“箭袖”称呼这种袖型，后来不用了。 @狐鸦youko 认为，“窄袖”也可以指袖根窄的袖子，所以称呼这种袖型为“箭袖”比“窄袖”更加合适。


同时，宋代亦尚无“箭袖”一词。“箭袖”是清代才出现并流行的词语。


针对直接用“飞机袖”当做形制代称的现象，微博@非常规养生博主古月与渊 认为，飞机袖是仅仅指袖子，不能指代整件衣服，因此不能用作形制的名称。


# “窄袖褙子”到底是不是褙子

褙子是真实存在的形制，但在汉服界的所指存在争议。


关于此形制，微博 @汉服买家咨询 认为，虽然有一种说法是从唐褙子到宋褙子之间可能出现过不同衣长的过渡，但是此形制称为褙子的依据不足，[6][7] 因此，该微博在发布此形制商品资讯时，标注为“宋制/直领对襟窄袖短衫（“飞机袖”）”。


考虑到这种形制的夹衣也可能存在（见前述“结构特征”与“文物依据”），或可视场合称呼为衫/袄。


# 是否可以对襟、交领两穿

很多商家将“窄袖褙子”做成对襟、交领两穿（汉服界俗称对穿交或对交穿），可能因为其轮廓容易使人联想到明代钱氏墓的另一件衫子。甚至有商家将“窄袖褙子”的下摆做成了明制的弧形下摆，这引发了一些争议。[8][9]


最先将“窄袖褙子”做成对襟、交领两穿的源头尚不明确。仅查考到微博@ 梅影詩魂·梅雪無名 微博[10][11] 称：


剛在洞娘看到妹子問宋代對穿交的情況，評論不讓我發圖，這邊附上報告截圖。南陵鐵拐墓的，窄袖那件數據成謎，寬袖那件不要乍看以為人家是短袖。那件衣長108，肩寬83。平舖83這數據放到漢子身上都顯大，更何況是妹子，只能推斷這是對穿交來著的。


则这条微博中所猜测的“宋代对穿交”，其实是同墓出土的另一件数据为“肩宽83”的“宽袖褙子”，不是“窄袖褙子”（即本词条中的形制）。其另一条微博尝试了用“窄袖褙子”衫作交领穿，但其本人在评论区也表示是推测穿法。[12][13]


一些壁画中，宋制女装长上衣的衣襟会出现小面积的自然相掩，可能是汉服界“宋制对穿交”猜测的来源。


但这是否是系带造成？笔记中虽然有男装背子腋下垂带两条的记载，但同时强调垂带仅用作装饰，并不用来束衣，因此系带可能并不是产生交叠的原因。[14]

壁画等艺术表现中，女子直领上衣的衣襟会有一部分自然相掩（图为白沙宋墓壁画东壁乐舞图，有时也称为开芳宴。图上画圈与意见来自微博 @大宋摩登bot）


并且，褙子腋下带有垂带目前仅限于男装。目前暂无依据证明宋制女装对襟衫的两襟是像明制那样用系带作交领连接的。南陵铁拐宋墓出土的此文物也并无系带残留。


有人从自身制衣经验出发，表示宋制衫袄通常没有锥度，如果大放量对襟作交领穿，又使用衣掩裙穿法，视觉效果不佳。宋制男装交领褙子/交领衫因为衣领为斜领、衣长较长、放量较大，可以缓解无锥度导致的视觉上的不和谐感。[15]


因此，目前此形制作为“对穿交”，或在衣领、腋下缀带以作交领穿，暂无依据，仅属于民间推测的范畴。
