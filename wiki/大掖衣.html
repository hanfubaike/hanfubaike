# 大掖衣

# 基本信息

  • 名称：大掖衣
  • 别名：缝掖、逢掖、逢腋、冯翼、逢衣
  • 特征：类似襌衣，而衣袖宽大，具体结构尚不明确。
  • 流行年代：汉代以前，至少春秋时已有
  • 文物依据：暂无服饰实物出土
  • 分类：暂不明确。暂定为：服-体衣-外衣-深衣-袍

河南信阳长台关1号墓出土瑟漆绘残片彩色摹本。有猜测中间这位楚国男子穿着的即是大掖衣。

图片来源：汤池, 信阳楚墓锦瑟漆画简介,美术研究,1980(3).


上述锦瑟实物图，出自《河南信阳楚墓出土文物图录》。



# 定义

一种古代儒生、士庶之服，衣袖宽大，亦称“逢掖”“缝掖”“逢腋”“冯翼”“逢衣”。[1]


逢、大同义。逢亦作缝。掖、腋同义。


# 结构特征

结构特征尚缺少服饰实物佐证。据文献记载，类似于衣袖宽大的襌衣。


# 艺术表现

河南信阳长台关1号墓出土瑟漆绘残片[2] 上绘有一楚国男子，衣袖宽大。有人认为即是“大掖衣”。

# 文献记载

* 《礼·儒行》：鲁哀公问于孔子曰：“夫子之服，其儒服与？”孔子对曰：“丘少居鲁，衣逢掖之衣。长居宋，冠章甫之冠。”


汉郑玄注：逢，犹大也。大掖之衣，大袂襌衣也（襌dān，音单）。此君子有道艺者所衣也。


唐孔颖达疏：掖谓肘腋之所，宽大，故云“大袂襌衣”也。

* 《列子·黄帝》：汝逢衣徒也。


张湛注：逢，犹大也，谓大掖之衣。

* 《墨子·公孟》：缝衣博袍。（按：一作“绛衣博袍”。）

* 《庄子·盗跖》：缝衣浅带。

* 明张岱《夜航船·衣裳》：逢腋，肘腋宽大之衣，为庶人之服。


# 大掖衣≠宋制“飞机袖”衫/袄

孟晖《中原女子服饰史稿》中解说河南白沙宋墓壁画上褙子的阔袖根、窄袖口的袖型表现时，用早期的“大掖衣”一词来形容它。因此在汉服界产生了误解，以为阔袖根、窄袖口的褙子即是“大掖衣”，进而认为安徽南陵铁拐宋墓出土的宋制“飞机袖”衫/袄的名称是“大掖衣”。而事实上大掖衣是汉代以前的服饰，原文已详。[3] 至于宋制“飞机袖”衫/袄是否白沙宋墓壁画上表现的褙子，详见宋制“飞机袖”衫/袄(窄袖褙子)词条。



# 注意

注意：本词条中所述汉服形制，踩中了如下三条中的一条或几条：

（1）缺少服饰实物依据，可供复原研究的资料过少

（2）存疑点过多

（3）争议点过多


因此，相应的“复原款”仅建议作为某一时代服饰风貌意韵的模拟，或古装剧、舞台服设的参考存在，未来或许会被考古新发现推翻，选购前请务必知悉风险。

如您掌握此形制的新资料，请通过编辑词条或讨论版帮助我们更新此词条。
