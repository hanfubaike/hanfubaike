# 中衣、裤、裙

# 基本信息

  • 名称：中衣、中裤、中裙
  • 分类：服-体衣-衬衣
  • 特征：中衣没有一定的特征，主要表示衣物的层次，因场合不同而选用不同材质、颜色和款式。
  • 流行年代：长期
  • 别名：衷、衷衣、亵、亵衣（有歧义）


# 定义

  1. 古时穿在祭服、朝服里面的衣服，但又在贴身内衣之外，介乎大衣（外衣）和小衣（内衣）之间，故名。
  2. 贴身的衣裤。[1]
    汉服界使用“中衣/裤/裙”一词，通常是指外衣之下，内衣之上的层次穿着的衬里衣物。



# 主要种类

中衣没有一定的特征，主要表示衣物的层次。因场合、搭配、时令的不同，而选用不同材质、颜色和款式。


# 文字记载

《说文·衣部》：衷，里亵衣也。段玉裁注：亵衣有在外者，衷则在内者也。


《礼记·郊特牲》：台门而旅树反坫，绣黼丹朱中衣。


孔颖达《正义》：中衣，谓以素为冕服之里衣。


《史记·万石张叔列传》：为郎中令，每五日洗沐归谒亲，入于舍，窃问侍者，取亲中裙、厕牏，身自浣涤。颜师古注：中裙，若今言中衣。


三国魏·繁钦《定情诗》：何以结愁悲？白绢双中衣。


《红楼梦》第三十四回：袭人听说，便轻轻的伸手进去，将中衣褪下。


# 关于衷、亵衣

衷是“中衣”的合字。也有写作“衷衣”的。据《说文解字》，“衷，里亵衣也”。段玉裁据此将衷解释为比亵衣更内一层的衣物，相当于本wiki分类中的“内衣”的层次，也就是义项2中“贴身的衣裤”之谓。然而检文献，“衷”也可以表示将某种衣物用作衬衣的意思，与义项1相近。例如：


《谷梁传》：或衷其襦。


《左传·襄公二十七年》：楚人衷甲。


宋·程大昌《演繁露·背子中禅》：今人服公裳必衷以背子。


亵衣的情况与衷类似。《说文解字》“亵，私服也”。表示不适合外穿见客的衣物。在具体使用中，经常用来表示较为私密的场合中的穿着，例如：


晋·葛洪《抱朴子·疾谬》：汉之末世，则异于兹：蓬发乱鬓，横挟不带，或亵衣以接人，或裸袒而箕踞。


明·沈德符 《野获编·佞幸·士人无赖》：兵部尚书王琼，头戴罛剌，亵衣，潜入豹房，与上通宵狎饮。


《扶风传信录》附录《后会仙记》：柳乃拂衣慢立，欲行礼，仇以亵衣辞，柳亦敛衣退，似无复拘拘者。


亵衣可能的层次，参考《说文解字》“衷”字条，如果“里亵衣”在层次上对应的是内衣，那么“亵衣”对应的就应该是中衣。


但是，在具体的使用中，亵衣也可以表示贴身的衣物。[https://baike.baidu.com/item/亵衣 某些百科]认为亵衣即古代女子的内衣之称，也有人认为亵衣即胸罩，这虽然是片面的，但这种用法确实存在。例如：


明·沈德符《野获编·内阁三·籍没二相之害》：其妇女自赵太夫人而下，始出宅门时，监搜者至揣及亵衣脐腹以下。


此外，亵衣在表示内衣时，可以是上半身的胸罩，也可以包括下体穿着的贴身衣物。例如：


茅盾《子夜》三：她的紧裹着臀部的淡红印度绸的亵衣，全都露出来了。


则中衣、衷、亵都兼有内衣、衬衣两个义项，在阅读文献时，要结合上下文推断其在文中的含义。
