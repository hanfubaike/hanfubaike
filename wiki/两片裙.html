# 两片裙

# 基本信息

  • 名称：两片裙
  • 别名：旋裙（据文献推测）、赶上裙（据文献推测，从表述看形态与出土文物有所不同）、上马裙（明代文献中的称呼，不可靠）两幅裙（德安周氏墓清理简报用语）、镶花边裙（黄昇墓报告用语）、两片直裙（部分论文、专著用语）
  • 分类：服-体衣-外衣-裙（裳）、裤-裙
  • 特征：围系、两联式、裙片不打褶、高腰
  • 流行年代：宋
  • 文物依据：福州南宋黄昇墓等广泛出土

# 定义

宋制裙装的一种，又名两片直裙。裙身为两联式结构，裙片不打褶，裙腰高，腰两端有系带，围系穿着的裙装。可以是单裙，也可以是夹裙。


可以与各种衬裤搭配，上搭衫子、褙子，用作便服（非正式场合）、常服（正式场合）的外衣裙装，不用作大礼服（特别正式的场合）。[1]


“两片裙”是现代命名，古代文献中无此称呼。其原型是许多宋代女子墓葬中均有出土的一种裙装。有学者根据其特征，认为此即是宋代文献记载中的此处为语雀文档，点击链接查看：https://www.yuque.com/hanfubaike/wiki/zhg99q。或谓是此处为语雀文档，点击链接查看：https://www.yuque.com/hanfubaike/wiki/opeo3b。

。


注意：两片裙不是指俗称“两片式”的双裙头齐胸裙，后者非汉服，详见“双裙头”。


# 结构特征

这种裙装为典型的两联式结构，上下两片裙面中间相互交叠，走路时腿部不外露，会自然显示出开胯效果。这种结构，是都城的女性艺人为了方便乘马乘驴而发明，而后风靡了士人阶层。这种独特的两联式结构使它和后来流行的马面裙显示出某种亲缘关系。[2]


从出土文物来看，左幅压右幅、右幅压左幅都是可以的，没有一定的规定。[3]


裙腰高，腰两端有系带，围系穿着。


这种裙子裙片并不打褶，裙摆较小，因此穿着效果类似于时装包臀裙，有垂坠感。宋代绘画中女子身材廓形通常较为修长纤细，可能这种裙装也代表了当时的审美好尚。裙装的腰宽总是小于下摆宽。目前对于文物的了解缺少公开的资料印证，有人认为是梯形拼接导致，也有人认为文物裙腰收有省道。[4][5][6]


从质地来看，两片裙文物多用绫、罗制成，二者均纹理较为致密，唯一绢裙是双层绢，可见这类裙装当时不以薄透为美。[7]


花边装饰不是两片裙的必要条件。黄昇墓大量两片裙缀有花边，德安周氏墓的没有花边。


# 文字记载

江休复（江邻几）《（嘉祐）杂志》：妇女不服宽裤与襜，制旋裙，必前后开胯，以便乘驴。其风开于都下妓女，而士大夫家反慕效之，曾不知耻。


《宋史•五行志三》：理宗朝，宫妃系前后掩裙而长窣地，名“赶上裙”。


明田汝成《西湖游览志余》卷二《帝王都会》：理宗时，宫中系前后掩裙，名曰上马裙。


赶上裙、上马裙的说法可能是指一种从旋裙基础上增加裙长发展而来的裙子，详见后文“争议”部分。


# 文物依据


# 江西德安南宋周氏墓出土

据《清理简报》，出土夹裙11件，其中罗质5件，绢质4件，绫质2件。有素色的，也有提花和印花的。提织花纹有山茶、朵云、桂花；印花有印金小瓣菊花和团花。内层多为绢和纱质。多为两幅相压，右幅压左幅。除去有2件单幅的（其中1件幅较窄，一片作成；另一件为褶裥裙，腰部有褶）共出土9件两片裙。[8]


德安南宋周氏墓出土两片裙代表性文物：


印花罗裙，裙长87.5cm，腰宽140cm，下摆宽155cm。


德安南宋周氏墓中共计出土了15条裙子，但仅有3条明确指出其形制，其中两片式裙仅有印花罗裙一条（《江西德安南宋周氏墓清理简报》中称为印金彩罗裙，根据图片信息应为同一条）。根据清理简报，所有裙子大体数据范围为长83~93cm，腰宽100~133cm，下摆宽116~146cm。[9]


# 福州南宋黄昇墓出土

黄昇墓共出土“单裙”17件，多罗地，上端略窄，4片分缝作两块，然后对错叠接成3片的宽度，居中错叠的宽度大于两侧，顶端合缝，下摆不缝，上部接裙腰，裙腰两端系裙带。裙两侧、下摆及缝脊多数缀有彩绘花边或印金填彩花边一道为饰。[10]


其中两片裙代表性文物：


浅褐绫四幅两片直裙，黄褐色提花山蔷薇，菊花纹绫缝制，镶彩绘菊花花边，裙长84cm，腰高12.7cm，腰宽122cm，下襕宽126cm，裙上片宽91cm，裙下片宽92cm，裙缘花边宽2cm，裙带左长79cm、宽 4.5cm，裙带右长37cm、宽4.5cm。[11]


黄褐色绢印靛蓝小点花裙，用蓝靛染料在黄褐色绢地上印小点，裙长86cm，腰高11.5cm，腰宽116.5cm，下襕宽126cm，裙带左长80cm、宽4.2cm，裙带右长64cm、宽4cm。[12]


此外还有夹裙一件，表里均为平纹绢，形制与单裙相同。


# 争议


# 两片裙是否可以用两幅长方形布重叠而成

部分商家使用长方形布重叠制作两片裙。有人对此提出不同意见，认为两片裙不应该仅仅是两幅长方形布上下重叠而成，因为文物普遍上下宽数据不等，存在斜裁拼接或收省的可能性很大。由于资料较为缺乏，黄昇墓两片裙结构示意图[13] 上所画疑似是梯形和长方形的组合，影响较大。但目前暂无公开讯息佐证，仅有人传闻文物是斜裁。与长方形拼接，一般认为是古代布幅不够宽，无法用整幅布制作导致的，目前的纺织条件下已经可以用整幅布来做。但考虑到人体形态，直接用长方形布重叠制作的裙子，不够修饰身材，视觉上也较为单调，因此在制作宋制无褶裙装时，有部分商家仍然会采用拼接工艺。


此外，“古墓仙女派”认为每片都使用了收省工艺，但是暂无公开讯息佐证。相关讨论参见[14][15][16][17] 。


# 两片裙是否就是“赶上裙”

“赶上裙”说出自《宋史·五行志》，是理宗时宫中的时尚，其中“前后掩裙”的说法使人联想到两片裙的文物形态。黄昇墓出土大量两片裙，墓葬是淳祐三年，早于理宗朝。且记载中称赶上裙“长窣地”，是一种拖地的长裙，与文物数据反映的情况不甚吻合，可能是一种在旋裙基础上增加裙长而产生的新时尚。从上下文来看，赶上裙在当时属于奇装异服。《西湖游览志余》中“上马裙”文字与《五行志》中“赶上裙”描述如出一辙而名不同，可能二者具有相同的文献来源。


# 两片裙是否用作褶裥裙的衬裙（中裙）

黄昇宋墓中，出土时黄昇身上穿着一条两片裙，下身从外到内顺序为褶裥裙、镶边裙、裤装、围兜、无裆裤、围件。高晨瑞认为，在黄昇身上，两片裙可能是作为褶裥裙的内衬穿着。但也指出，德安周氏墓报告没有类似搭配信息提及，而花山墓没有旋裙发现，可见二者并没有形成捆绑固定的穿着体系。[18]


张玲《南宋淳祐时期贵族妇女着装形态考释》认为，褶裥裙是大礼服（上装最外层为大袖衫、霞帔的场合）中使用的裙装下装。而两片裙（文中称为单裙）是常服（上装最外层为背子的场合，原文称为正式常服）、便服（上装最外层为衫袄，原文称为非正式常服）的裙装下装。不论哪一种裙装，都要与开裆裤（袴）、合裆裤（裈）搭配[19] 。考虑到不止一座宋墓的尸身装殓都有穿着层次过多，疑似将所有场合的服装都穿在身上的倾向，这种推断是有一定道理的。
