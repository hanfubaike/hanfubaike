# 为什么两片式不是汉服

两片式指裙头分为两片的一种非汉服，详情参考“两片式”词条。


关于此问题，历来讨论者多。微博 @齐衡家的申和珍 综合前人观点，撰《关于为什么"两片式"不是汉服》一文，论之最详，现转载如下，作为这一问题的解答。


# 关于为什么"两片式"不是汉服


# 两片式的由来



2009年有商家在贴吧上发表文章，试图复原唐制齐胸衫裙，因壁画上的仕女裙有侧开叉的样式，进行了推测，试图借鉴和服里行灯袴的两片式裙头来还原唐裙，引起一众商家开始模仿，在近两年形成热潮。

由此，提出第一点质疑——


# 和服行灯袴究竟能不能作为借鉴依据？

——答案是不能。和服行灯袴的起源于日本平安年年间，盛行于明治时期（也就是中晚清时期），保存在现代作为女子毕业服使用，年代与唐裙不符。服饰的演化都是向前发展的，且和服早在唐代衰败之后就脱离汉服独立发展了。因此这个论据显然站不住脚


第二点质疑


# 难道我们自己不会演化出这种形制的裙子吗？

——我的答案是存疑。首先，隋唐以前并没有史料记载过、出土过具有明显裙头分片的裙，唐之后更没有这种结构的存在，无论是史料、文物、艺术形象哪一方面，两片式都是从来未曾有过姓名，推测其曾经存在显然站不住脚。第二，有人可能说，可能它就流行过一段时间就消失了呢？符合这个定义的，我们通常称为“服妖”，即因为种种因素突然出现又突然消失的当时的奇装异服，且只有极个别人穿。因此“服妖”是无法满足“汉民族传统服饰”这个定义的，因为它不具有普遍性。从这个角度说，两片式依旧不是汉服。


题外话：如果对唐裙的复原感兴趣，可上微博搜索@—醉花涧—，她研究实验出了新的唐裙的结构模式

# 两片式能否作为改良汉服？


改良的在搜狗百科上的定义是“改良，英文可译为improve，指去掉事物的某些缺点，使之更适合要求或者改善。”那么两片式有哪些良呢？恕我直言，除了转圈圈好看之外，它恐怕再无优点。


第一，  分前片后片，胸小的妹子根本穿不了，前片会掉，就算前片不会掉，后片由于背部没有支撑也会滑掉（这一点上大胸小胸的妹子是平等的）；


第二，  腋下有两个缝，和第一点一样，缺点是容易走光；


第三，  长度和复原汉服一样长，这一点上并没有改良；


第四，  部分商家和买家为了追求好看做出九米摆、十二米摆、十五米摆，恕我直言，只是为了拍照好看而做出来的不实用不日常的衣服本来就是影楼装。


综上，两片式并没有改良，不能作为改良存在


# 为什么现在两片式的存在不合理？

民族传统服饰为何称为传统，核心就在于传承。而衣服的剪裁方式更是传承的主要内容。从秦汉到晚明，形制再怎么改，上衣的中缝从来没有歪过，右衽也从来没有废除过，这是传统的核心，而下面的裙子也应该一样，具体的制式再怎么改，一片裙头也不应该被动摇。所以，两片式可以类比没有中缝且左衽的上衣一样，不应该被称为汉服。


最后，我想谈谈形制，形制是传统服饰的根，就像偏远地区的苗族白族生活再困难也不会舍弃掉头上的传统银饰（没有歧视的意思），服饰文化应该作为传统文化的一部分被慎重对待，而不是被当做小众文化张扬个性。我希望有些人对我们这个民族保有一份尊重，审美是很个人的事情，但形制是整个民族的事情。你觉得好看你去穿没问题，衣服没做错什么，我们反对的从来只是指鹿为马的行为。至于那些认为自己受到了排挤，被欺骗的人，你们的矛头从来都不该指向努力研究努力科普的人，而是那些打着汉服名头贩卖情怀欺骗你的商人。[1]
