# 抹胸

# 基本信息

  • 名称：抹胸
  • 别名：抹胸肚、袜（mò）、袜肚、袜腹、兜肚（注意：使用不同别名时，内涵或有所差别）
  • 分类：服-体衣-内衣
  • 特征：内衣
  • 流行年代：长期
  • 文物依据：南宋花山墓出土菱形朵花纹印花绢抹胸等


菱形朵花纹印花绢抹胸，南京花山宋墓出土，图片来自南京市博物馆。


绢抹胸，福州南宋黄昇墓出土，形态与花山墓所出土的有明显不同。


商家仿宋制抹胸，图片来自云绣楼


男式梯形抹胸，江苏金坛南宋周瑀墓出土，图片来自黄能馥、陈娟娟《中国服饰史》。



# 定义

裹胸使之收束固定的内衣的通称。古代典籍中，“抹胸”一词形成得较晚，但裹胸内衣很早就有了。一些汉服商家习惯将抹胸称为“宋抹”，但抹胸并不是宋代才有的，也并非仅流行于宋代，只是因为这类商品最初是仿照目前出土的一种宋代抹胸制作的。


历史上，抹胸不仅用于女子，还被男子使用。[1] 不仅汉族穿着抹胸，其他民族也有穿着抹胸类衣物的记载。[2]


主腰即明制抹胸的别名，详见“主腰”词条。


# 结构特征

目前商品化的抹胸，主要是以柔软亲肤的面料制成，有单层、双层，也有双层可双面穿的。


通常是一条长度合围或超过胸围、宽度可从乳上覆盖到腰际及以下的布片，上缘两端缀以扎束用的布条。穿着时，用布片长边的正中央对准身体前方正中央，两边向后缠裹身体，将布条在胸前扎束，之后将布条上方的布料下翻，起到加固作用。


还有一种四周系带的抹胸，只施于胸，不施于背，类似肚兜，是仿制福州南宋黄昇墓的出土文物设计制作的。


# 主要种类


# 合围式抹胸

在基本结构的基础上，合围式抹胸主要有三种款式：

  1. 普通一片式抹胸。在布片部分除正常锁线锁边外不做任何额外的打褶和收省。此为一式。
  2. 一片式工字褶抹胸。为使抹胸不至于滑落，通常会设法在抹胸上缘制作出曲度。常见的做法是在上缘中央附近打一个工字褶。此款式目前已经有一定的流行度，但做法有争议，恐是早年对出土文物资料的掌握、观察、了解不充分导致的误解。事实上目前的抹胸文物胸前的褶皱是一个三角形的收省，而未有工字褶抹胸的实物面世。此为一式。
  3. 一片式收省抹胸。即仿照文物的形貌，在中央做一个三角形的收省，起到贴合胸部曲线的效果。此为一式。



# 四周系带抹胸

只施于胸，不施于背的抹胸，又名“袜（mò）肚”，是一片布，四周缀以上方两根，左，右各一根，共四根系带。上方两根系带先垂至身后，两侧系带身后交叉，绕到身前系结。然后将上方系带穿过背后横向系带，打结固定。


# 改良抹胸

改良抹胸，通常是普通吊带背心，或在普通吊带背心基础上，于胸前做出工字褶，模拟出一种古朴的效果（工字褶有争议，详见“合围式抹胸”第二式）。


# 文字记载

抹胸在古代文字记载中常被称为“袜”（mò），和袜子的“袜”是同形字，是汉字简化导致的，熟习当代简体字的朋友尤其需要注意。古代文字记载中，“袜”不是今天理解的袜子（即足衣），而是胸衣，繁体仍然写作“袜”。而作为足衣的袜，往往写作“襪”。


刘孝标乐府：裲裆双心共一袜，袹腹两边作一襊。


《广韵》：袜肚。


《类篇》：所以束衣也。


隋炀帝诗：锦袖淮南舞，宝袜楚宫腰。


《辽史拾遗》卷十五：张舜民《画墁录》曰：“契丹待南使，乐列三百馀人，舞者更无回旋，止于顿挫，伸缩手足而已。”“角抵以倒地为负，两人相持终日，欲倒而不可得。”又“物如小额，通蔽其乳，脱若褫露之，则两手覆面而走，深以为耻也。”


《金瓶梅》第七十五回：又向床头取过她的抹胸儿，替她盖着胸膛。……妇人道：“这衽腰子，还是娘在时与我的。”


《红楼梦》：这尤三姐索性卸了妆，脱了大衣服，松松挽着头发，身上只穿大红袄子半掩半开，故意露着葱绿抹胸，一痕雪脯。


徐珂《清稗类钞 服饰》：抹胸，胸间小衣也。一名袜腹，又名袜肚。以方尺之布为之，紧束前胸，以防风之内侵者。俗谓之兜肚。


《上海市大观》：女人衣饰中，有一神秘之物，使男子见之而魂销者，则抹胸是也。抹胸之名称甚多，苏人则呼之曰肚兜。因为妇女紧束前胸，以护其乳，辄令男子作艳想。骚人词客，恒有以此为吟咏的，什么“塞上酥凝，峰头玉小”，风魔了许多色情狂。抹胸是作斜方形的，上系以绳，环诸颈间。年青的女人，往往系以金链，这种金链条，有粗有细各随她们的喜欢。中等人家的女子，不用金链条，也有用一根银链的。即小家碧玉，也有用一根朱红绳的，在雪白的粉颈上，露出一缕红痕，也使人发生一种艳想。下面又有两条带，束在背后，而最下的一只尖角，直遮过肚脐，达于小腹。所以这抹胸是女子最接近肌肤之物了。抹胸的颜色，种种不一，自然以颜色最娇艳为最佳。质料是夏日以纱，冬日以绉，总以轻适为宜。红色的抹胸，尤为人所艳称，因为有刺激力。更有在抹胸上洒以香水，贮以麝屑的，这种媚人之法，真做到了“罗襦乍解，微闻芗泽”了。其实妇女在抹胸时代，倒也宽紧随意，并不束缚双乳。自从流行了小马甲以后，那真与卫生有害，足以戕害人体天然的生理。[3]


# 诃子是抹胸吗？

有一种流传已久的说法是唐代的女性内衣叫做诃子，微博@洛梅笙 认为诃子不是抹胸，而是诃黎勒桃形果实制作的饰物，略论如下：


诃子这个词作为抹胸很奇怪，因为在唐代似乎不见，最初仅在北宋的《事物纪原》里有说到诃子，里面的记载是这样的：本自唐明皇杨贵妃作之，以为饰物。贵妃私安禄山，以后颇无礼，因狂悖，指爪伤贵妃胸乳间，遂作诃子之饰以蔽之，事见《唐宋遗史》。就是说杨玉环和安禄山私通，然后把杨玉环的乳房中间的地方抓伤了，所以就拿诃子遮蔽，并说这条出自《唐宋遗史》，这本书据说是一本收集各种野史秩闻的笔记，看了《撒马尔罕的金桃》，个人有一个想法，这个故事很可能是来自安史之乱，也就是唐代衰落时期的某个脑洞，事实上本草纲目中就有提到什么是诃子，即原产印度的诃黎勒（梵文名），诃黎勒是在开元天宝时代由外国使团进贡的一种植物，当时进献的是三勒，即庵摩勒，毗黎勒和诃黎勒，鉴真东渡就从中国带去了诃黎勒等药材，今天正仓院还藏有八世纪保留下来的诃黎勒，这种桃形果实，至少在事物纪原的时代，人家也没有认为他是抹胸，而是一种饰物，到了明末清初的尤侗，他写过一首诗：湖罗三尺海棠红，结束层层闭上宫。妃子金诃藏个里，姬人宝镜挂当中。青牛帐底常凝露，朱鸟窗前却避风。何日与郎亲解取，最堪消渴玉酥融。这里的诃子则发展成了金诃，但他的意思其实和《事物纪原》差不多，并没有认为诃子是抹胸，直到了晚清时代的叶炜的《煮药漫钞》，意思逐渐扭曲了，叶炜生于道光十九年，死于光绪二十九年，他在《煮药漫钞》中说：妇人袜胸，制自杨太真。因安禄山爪伤杨妃乳，乃为袜胸蔽之，名“金诃子”。则完全自编自导了一个新版本，首先，女人的抹胸是杨贵妃设计发明的，其次，制作的目的是杨贵妃的乳房被抓伤，所以设计了抹胸，叫金诃子。但如果对照一开始的说法，其实要表达的意思已经相差甚远。为什么会出现这样的误区，一种最大的原因大概就是，在叶炜所处的晚清，女性的内衣已经发生了很大的变化了，尤侗的时代他不会将诃子误会成抹胸是因为当时那种以长巾束胸是一种常态，包括前襟带扣子的主腰等也都是大家日常知道的东西，而到了清末近代，女性开始流行小马甲的内衣的时代，叶炜无论是出于猎奇也罢还是经验常识的缺失，都让人对古人的说法不明其义。[4]


# 文物依据


# 合围式抹胸

菱形朵花纹印花绢抹胸， 南京花山宋墓出土，中央有收省，花纹精美。长方形，长50cm，宽116cm，两边系带分别为长51cm，宽3.5cm，上部中间处打有一个折褶，褶宽5cm，深10cm。面料为1/1平纹绢，以印花工艺用黑色印出菱形纹，菱纹内填以柿蒂小花，相错排列，花纹循环，经向1.5cm，纬向1.5cm，衬有绢里，1/1平纹组织，并以宽6cm的朵花纹印花绢贴边。


素纱抹胸，南京花山宋墓出土。长方形，长46cm，宽112cm，两边有系带，带长61cm，宽3.5cm，中央打有一个小褶，褶宽5cm，深10cm，织物组织为1：1绞纱，系带为1/1平纹绢。


芙蓉山茶梅花纹罗抹胸，南京花山宋墓出土。长方形，长50cm，宽116cm，两边系带长59cm，宽3.5cm。中央有一折褶，褶宽4cm，褶深10cm。面料地部组织为1：1绞，花部组织为1/2左向斜纹花，图案是一排芙蓉、一排山茶、一排梅花交替排列，花为折枝。图案循环，经向10cm，纬向8.5cm。上下两头存幅边，通幅正好是抹胸的长度，系带织物组织为1/1平纹绢。


卍字菱格纹罗抹胸，南京花山宋墓出土。长方形，长50cm，宽120cm，破损严重，多处糟朽，系带脱落，分别只剩下5cm，1cm的长度。面料组织结构地部为1：1绞三梭横罗，花部位1/2右向斜纹。图案主体是菱形，在菱格内为一个卍字，菱格外为联珠形成的菱形，四方连续排列，图案循环，经向7cm，纬向4.5cm。


素绢抹胸，南京花山宋墓出土。长方形，长42cm，宽90cm，另有一块长92cm，宽13.5cm的织物应是该抹胸的上半部分横向系带，现已与正身脱线分离，尚能看出残留的系带断头，与一根系带断裂系带正好可以对位连接，另一根系带缺。面料组织结构为1/1平纹绢。此抹胸中央未见收省。

南京花山宋墓出土菱形朵花纹印花绢抹胸的花纹，图片来自微博@撷芳主人


素纱抹胸，南京花山宋墓出土。长方形，长40cm，宽80cm，两边系带分别长为80cm，宽为16cm。面料组织结构为1：1绞纱。此抹胸中央未见收省。[5]


# 四周系带抹胸

素绢抹胸，南宋黄昇墓出土。以素绢为之，两层，内衬少量丝绵，长55cm，宽40cm，上端及腰间各缀绢带两条，以便系带，带长34-36厘米。[6]


素绢抹胸，江苏金坛南宋周瑀墓出土，为掩胸之衣，用驼色素绢对角缝制成梯形，上宽15cm，下宽83cm,，竖直30cm，下边中间打褶2厘米，上下钉有4根绢带，宽2.5，残存最长者63cm。[7] 此为男式抹胸。女子抹胸面积比男子抹胸大，不仅护胸，而且裹肚腹；两者皆有系带，以女子抹胸系带为多。[8]
